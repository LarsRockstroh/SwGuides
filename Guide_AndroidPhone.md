## Programs

* F-Droid
    * <https://f-droid.org>
    * FOSS app distribution ecosystem for Android
* Vanilla Music Player
    * <https://f-droid.org/packages/ch.blinkenlights.android.vanilla>
    * <https://vanilla-music.github.io/>
* K-9 Mail
    * <https://f-droid.org/en/packages/com.fsck.k9>
* Termux
	* <https://f-droid.org/en/packages/com.termux>
	* <https://termux.dev/en>
	* <https://github.com/termux/termux-app>
	* Storage
	    * Grant access to storage via Settings->Apps
	    * '/storage/emulated/0'
	    * Setup links: 'termux-setup-storage`
	* `pkg install python mc`
	* `pkg install mc`
* FairScan (Photos to PDF)
    * <https://f-droid.org/en/packages/org.fairscan.app>
    * <https://github.com/pynicolas/FairScan>
    * <https://fairscan.org>
    * Enable light for photo capture
* LibreOffice Viewers
    * <https://f-droid.org/en/packages/org.documentfoundation.libreoffice>
* MyPhoneExplorer


## Settings

* Check for updates (System, Security, Play) and install until no more updates available 
* Settings -> Apps: Uninstall not needed Apps
    * Fitbit
    * Galleryit
    * Family Space
* Settings -> Apps: Force Stop/Disable not needed Apps that cannot be uninstalled
    * Assistent
    * Games
    * Gmail
    * Google TV
    * Moto App Manager
* Settings -> Apps -> Cloud media settings: None
* Settipgs -> Display
    * Adaptive brightness
    * Auto-rotate screen
* Settings -> Battery
    * Battery percentage (in status bar): On
    * Overcharge Protection
